# tedchang-lecture-production

테드창(임창식) 마스터님의 왕초보 AI 강의 제작 방법론. 30년 강의 경험 기반.

## 핵심 원칙

1. **왕초보 대상** (50~60대 다수, 일부 20~30대)
2. **예·복습 제로** — 수업 안에서만 따라가면 끝
3. **사용 도구**: ChatGPT(무료) + Gemini(무료) — 중국산 AI ❌, Claude ❌ (무료티어 부족)
4. **총 2시간 (120분)** — PPT 15~16장 구성
5. **노트북LM 호환 포맷** — 복붙하면 PPT 생성 가능

## 강의 구성 템플릿 (120분 기준)

| 파트 | 시간 | 내용 |
|:----:|:----:|:------|
| 🎤 도입/복습 | 0~15분 | 지난 시간 복습, 10초 퀴즈 |
| 📚 1부 | 15~50분 | 핵심 실습 (따라하기) |
| 🎨 2부 | 50~100분 | 신규 기능 + 재미 요소 |
| 🎮 3부 | 100~110분 | AI랑 놀기 (게임·운세) — 선택 |
| 🎯 마무리 | 110~120분 | 오늘 배운 3가지 정리 |
