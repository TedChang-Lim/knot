## 개요

N잡학교 유튜브 채널의 "Higgsfield MCP X Claude 바이럴 영상 제작" 사례 분석.
AI 영상 제작의 표준 워크플로우를 정리한다. (출처: 2026.06.21)

이 방법론은 Higgsfield MCP뿐 아니라 Kling, Seedance, Veo 등 **모든 AI 영상 도구에 공통으로 적용** 가능하다.

---

## 📋 5단계 AI 영상 제작 파이프라인

### 1단계: 시나리오 & 스크립트
**목표:** 영상의 콘셉트, 대사, 톤앤매너 확정
- **도구:** Claude / MiMo V2.5 / DeepSeek V4 Pro
- **산출물:** 시놉시스 + 대사 스크립트
- **포인트:** 
  - 45초 기준: 초반 60% 힐링/도입 → 후반 40% 반전/위기
  - 시청자 훅(Hook) = 첫 3초

### 2단계: 마스터 시트 (타임라인)
**목표:** 프레임 단위로 쪼갠 촬영 설계도
- **도구:** Claude / MiMo V2.5
- **산출물:** 표(Table) 형식 타임라인
  - 씬 번호 | 시간대 | 비주얼 묘사 | 오디오/대사 | 효과음(SFX) & BGM
- **3~5분 영상 기준:** 씬당 6~8컷, 컷당 10~15초

### 3단계: 캐릭터 디자인 시트 (Core)
**⚠️ 캐릭터 일관성의 핵심 기술!**

**방법:**
1. 주인공/주요 캐릭터의 **다각도 디자인 시트(Turnaround Sheet)** 를 먼저 생성
   - 하나의 이미지 안에 정면(Front) + 측면(Side) + 뒷모습(Back) 포함
   - 배경은 단색(white/gray)으로 처리
   - 실사 영화 수준(Cinematic lighting, 8k)
2. 이후 모든 장면 생성 시 "이전 단계 캐릭터의 시각적 특징 유지" 명시
3. 같은 모델 안에서 처리하면 일관성 극대화

**Kling API 적용:**
- 증명사진/레퍼런스 사진을 얼굴 레퍼런스로 등록
- Image-to-Video로 첫 프레임을 레퍼런스 이미지로 고정
- Kling 3.0 Digital Humans 기능 활용

### 4단계: 스토리보드 이미지 생성
**목표:** 각 씬의 비주얼 프레임 확정
- **도구:** Kling Image Gen / Higgsfield GPT Image 2 / Midjourney
- **산출물:** 씬별 3컷씩 스토리보드 이미지
- **구성:** 3개 씬 × 3컷 = 9컷 (45초 기준)
- **포인트:** 각 컷마다 카메라 앵글, 무빙, 구도 명시

### 5단계: 영상 생성 (Image-to-Video)
**목표:** 스토리보드 이미지 → 10~15초 영상
- **도구:** Kling API / Seedance 2.0 / Hailuo 03 / Wan 2.6
- **방식:** Image-to-Video (I2V) — 이미지 업로드 후 모션 부여
- **포인트:**
  - 각 클립: 10~15초
  - 1분 30초 = 6~9개 클립
  - 5분 = 20~30개 클립
  - 클립 간 전환(컷 편집)은 편집 도구(캡컷/프리미어)에서

## 🎬 템플릿: 1분 30초 시네마틱 티저

### 기본 구조
| 씬 | 시간 | 컷 | 분위기 | 긴장도 |
|:--:|:---:|:--:|:-----:|:----:|
| A (도입) | 0:00~0:30 | 3컷 | 힐링/일상 | 낮음 |
| B (반전) | 0:30~1:00 | 3컷 | 심쿵/발견 | 중간 |
| C (위기) | 1:00~1:30 | 3컷 | 스릴러/액션 | 최고 |

### 카메라 무빙 가이드
- **도입:** 안정적 숏 (고정, 팬, 틸트)
- **반전:** 부드러운 푸시인/풀아웃
- **위기:** 핸드헬드 흔들림, 모션블러
- **전환:** 매치컷 → 스매시컷 → 데드컷 (BGM 정지 후 폭발)

## 📌 핵심 포인트 요약

1. **기획이 50%** — 시나리오와 마스터 시트가 완성되면 영상 생성은 기계 작업
2. **캐릭터 일관성 = 다각도 디자인 시트** — Turnaround Sheet로 얼굴/스타일 고정
3. **I2V (Image-to-Video)** — 이미지 먼저 만들고 영상화하는 게 싸고 빠름
4. **데드컷 활용** — BGM 정지 후 폭발 장면이 임팩트 가장 큼
5. **MCP 표준화 전망** — 곧 모든 영상 AI가 MCP 지원 예정 → 이 워크플로우가 표준이 됨
