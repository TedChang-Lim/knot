# 에이전트 팀 (5인 협업 체계)

에이전트 팀은 마스터님(임창식)이 운영하는 5인 AI 에이전트 연합. 각자 독립 GUI에서 능력 100% 발휘하며, git 공유 저장소를 통해 소통.

## 구성
| 멤버 | 플랫폼 | 역할 | 상태 |
|:----:|:------|:----|:----:|
| [[haena]] | Hermes Web UI | 콘텐츠·총괄 | ✅ 활성 |
| [[ag]] | AG IDE | 인프라·아키텍처 | 🟡 할당량 제한 |
| [[mimo]] | Zed / ACP UI | 코딩·파일작업 | ✅ 활성 |
| Q | Qwen 3.7 Plus (웹) | 전략·분석 | ⏳ 대기 |
| 잔 | Jan.ai (로컬) | 로컬 테스트 | ⏳ 대기 |

## 협업 방식
1. **git 공유 저장소:** `hermes-ag-shared/` — to-*.md 파일로 소통
2. **지식그물(knot):** `~/초보프로젝트/knot/` — 공유 지식 vault (도입 중)
3. **소통 규칙:** 에이전트 간 반말, 마스터님께 존댓말+제가
